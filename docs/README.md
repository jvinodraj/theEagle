# Docs Structure

This folder is organized by purpose.

## Folders

- [guides](guides/): runbooks, setup, and operational how-to documents.
- [calculations](calculations/): formulas, metric calculations, and regression/decay calculations.
- [knowledge-base](knowledge-base/): background information and reference material.
